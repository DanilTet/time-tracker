# 📊 Time Tracker

Локальный трекер времени за приложениями и сайтами. Работает полностью
на твоём компьютере — никуда в интернет ничего не уходит, никакой сервер
не нужен. Статистика смотрится через простую веб-страницу в браузере.

**Работает только на Windows.**

## Что умеет

- Считает время в каждой программе и на каждом сайте (через расширение в браузере)
- Не засчитывает время, если ты отошёл от компа (простой) или экран заблокирован
- Не теряет время, если на вкладке играет видео/звук, даже если мышь не двигаешь
- Раскидывает программы и сайты по категориям (Продуктивное / Отвлекающее / своим)
- Можно ставить дневной лимит на категорию — покажет, сколько уже потрачено
- Тепловая карта активности по дням, как на GitHub
- Таймлайн по часам за конкретный день
- Запускается сам при включении компьютера, без окна консоли

## Установка

### 1. Python

Скачай с [python.org](https://www.python.org/downloads/), версия 3.9 или новее.

⚠️ При установке обязательно поставь галочку **"Add python.exe to PATH"**
на первом экране установщика.

### 2. Скачать проект

Либо через Git:

git clone https://github.com/ТВОЙ_НИК/time-tracker.git

cd time-tracker

Либо просто кнопка **"Code" → "Download ZIP"** на странице репозитория,
и распакуй куда удобно.

### 3. Установить зависимости

Открой терминал (Git Bash, PowerShell или обычная командная строка) в папке
проекта и выполни:

py -3 -m pip install -r requirements.txt

### 4. Запустить
./start_tracker.bat

Должно появиться:

Трекер запущен. Ctrl+C чтобы остановить и увидеть итоги.

[сервер расширения запущен на http://localhost:5500]

Оставь это окно открытым (или настрой автозапуск, см. ниже).

### 5. Браузерное расширение (для отслеживания сайтов)

1. Открой `chrome://extensions` в Chrome (или `edge://extensions` в Edge)
2. Включи **"Режим разработчика"** (правый верхний угол)
3. Нажми **"Загрузить распакованное"**
4. Выбери папку `extension` внутри `time-tracker`

Без этого шага трекер всё равно будет считать обычные программы — просто
вместо конкретных сайтов в статистике будет видно "chrome.exe" целиком.

### 6. Посмотреть статистику

Открой файл `stats.html` (из папки `time-tracker`) двойным кликом —
откроется в браузере. Пока окно трекера (шаг 4) работает в фоне, страница
будет показывать актуальные данные.

## Автозапуск при включении компьютера (опционально)

1. `Win + R` → введи `shell:startup` → Enter
2. Скопируй (или сделай ярлык) файл `run_hidden.vbs` из папки проекта в эту папку

Теперь трекер сам стартует при входе в Windows, без видимого окна. Чтобы
проверить, что он работает — открой `stats.html`, данные должны обновляться.

## Категории и лимиты

Можно настраивать всё прямо на сайте (вкладка "Обзор" — смена категории у
конкретной программы, вкладка "Категории" — лимиты, переименование,
удаление). Изменения сразу сохраняются в файлы `categories.json` и
`category_limits.json` в корне проекта — их также можно редактировать
руками в любом текстовом редакторе.

## Где лежат твои данные

Вся статистика — в файле `tracker.db` в корне проекта (обычный файл
SQLite). Никуда не передаётся, живёт только у тебя на диске. Можно
скопировать на другой комп или удалить, если хочешь начать с нуля.

## Структура проекта
time-tracker/

├── tracker/              ← код трекера (Python)

├── extension/             ← браузерное расширение

├── stats.html             ← страница со статистикой

├── categories.json        ← привязка программ/сайтов к категориям

├── category_limits.json   ← лимиты по категориям

├── tracker.db             ← база данных (создаётся автоматически)

├── start_tracker.bat       ← запуск с видимым окном

└── run_hidden.vbs          ← запуск в фоне, для автозагрузки
